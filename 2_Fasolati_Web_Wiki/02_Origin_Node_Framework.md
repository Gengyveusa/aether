# Origin Node Framework: Mouth + Gut

> **Source:** fasolati.life -- Problem section, Platform section, About section
> **Framework:** Two origin nodes (Oral, Gut) feeding one systemic output metric
> **Position:** The scientific and architectural foundation of the Fasolati platform

---

## Core Concept

> "Fasolati is an AI-driven platform that treats the mouth and gut as origin nodes of systemic inflammation, wiring oral therapeutics, gut repair, and biomarker-driven protocols into a single operating system for human health."

The body is a connected system. Two sites -- the **mouth** and the **gut** -- function as controllable upstream origin points where chronic systemic inflammation begins. Fasolati treats them as one integrated network.

---

## The Two Origin Nodes

### Origin Node 1: The Mouth

| Property | Detail |
|----------|--------|
| **Primary pathology** | Periodontitis (chronic biofilm-driven infection) |
| **Mechanism** | Keystone pathogens (*P. gingivalis*, *T. forsythia*, *A. actinomycetemcomitans*) generate inflammatory signaling that enters systemic circulation |
| **Biofilm role** | Structured microbial communities create protected reservoirs of chronic inflammation |
| **Intervention products** | [[03_Lytica_Biofilm_Intervention]] (active intervention), [[04_Loria_Molecular_Purification]] (daily maintenance) |
| **Professional channel** | [[10_For_Clinicians]] -- Maxoral Rx-grade formulations |
| **Data inputs** | Clinical periodontal data, microbiome composition (salivary diagnostics), daily self-reported data |

### Origin Node 2: The Gut

| Property | Detail |
|----------|--------|
| **Primary pathology** | Gut barrier dysfunction (increased intestinal permeability / "leaky gut") |
| **Mechanism** | Compromised barrier allows endotoxin (LPS) leakage into systemic circulation, driving chronic low-grade inflammation |
| **Microbiome role** | Dysbiosis -- loss of keystone commensals (*Akkermansia*, *F. prausnitzii*, *Bifidobacterium*) impairs barrier integrity and butyrate production |
| **Intervention products** | [[05_Gut_Restoration_Stack]] -- Magisnat, Biolumen, Monch Monch |
| **Biomarkers** | Zonulin (tight junction regulation), calprotectin (intestinal inflammation), LPS-binding protein (endotoxin leakage) |
| **Data inputs** | Barrier function biomarkers, microbiome diversity metrics, daily input data |

## Key Principle: Treat the Network, Not the Symptom

> "The body is a connected system. Chronic disease isn't a collection of independent failures -- it's a network of interconnected nodes where upstream dysfunction propagates downstream."

Fasolati's approach differs from conventional medicine in that it does not treat each disease endpoint independently. Instead, it monitors and intervenes at the two origin nodes, measuring the systemic output to validate that origin-level intervention is reducing downstream risk.
