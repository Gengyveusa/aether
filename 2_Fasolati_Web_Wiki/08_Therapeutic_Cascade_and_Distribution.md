## The Therapeutic Cascade

> "Lytica Hunts. Loria Purifies. The cascade is the system of care."

The Fasolati therapeutic cascade is a **two-step oral protocol** designed to first clear active disease and then maintain ecological health indefinitely.

### Step 1: Lytica HUNTS

| Action | Description |
|--------|-------------|
| **Clear the active infection** | Stage 1 BREACH: enzymatic matrix degradation |
| **Breach the biofilm** | H2O2, DNase I, Dispersin B, Proteinase K |
| **Eliminate keystone pathogens** | Stage 2 HUNT: bacteriophages, endolysins, lactoferrin |
| **Reset the ecology** | Restore conditions for commensal recolonization |

### Step 2: Loria PURIFIES

| Action | Description |
|--------|-------------|
| **Maintain the ecology daily** | Selective antimicrobial baseline (CPC 0.05%) |
| **Disrupt biofilm reformation** | Enzymatic polysaccharide scaffold dispersal |
| **Barrier, heal, hydrate, hold** | Dual-weight HA + castor oil + pullulan/PVP |
| **Indefinitely** | Daily use with no long-term adverse effects |

## Two-Tier Distribution Architecture

Fasolati products reach market through **two distinct channels** with different regulatory profiles, concentrations, and economics.

### Tier 1: Maxoral (Rx Grade -- Professional Channel)

| Property | Detail |
|----------|--------|
| **Grade** | Prescription-strength |
| **Formulations** | Higher enzyme concentrations, full phage payloads (when cleared), extended-release tray application |
| **Target users** | Periodontists, oral surgeons, DSO networks |
| **Distribution partners** | Henry Schein, Patterson, Benco |
| **Revenue model** | High-margin per-procedure intervention |
| **Regulatory** | Rx required |

### Tier 2: Fasolati (OTC Grade -- Consumer Channel)

| Property | Detail |
|----------|--------|
| **Grade** | Consumer formulations optimized within OTC regulatory limits |
| **Formulations** | Daily Loria maintenance, Stage 1 Lytica enzyme gel |
| **Target users** | Health-conscious consumers, post-treatment maintenance patients |
| **Distribution** | D2C (fasolati.life), Amazon, pharmacy, subscription |
| **Revenue model** | Recurring subscription revenue stream |
| **Regulatory** | OTC -- no prescription required |
