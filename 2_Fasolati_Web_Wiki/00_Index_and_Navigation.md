## Platform Architecture

- **Type:** Single-page application with overlay panels and smooth-scroll navigation
- **Rendering:** Client-side (JavaScript SPA with intersection-observer reveals)
- **Visual:** Particle canvas background with mouse-following interactivity
- **Animation:** Staggered reveal animations (.mo class with delay variants .mo-d1 through .mo-d4)
- **Forms:** HubSpot Forms API integration (portalId: 243149587)
- **Data:** localStorage backup for form submissions, cookie-based tracking (hubspotutk)
- **Interactive elements:** Collapsible framework nodes (toggleNode), expandable pills (togglePill), dropdown panels with 300ms hover delay
