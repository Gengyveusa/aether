## The Inflammation Opportunity

| Metric | Value | Source |
|--------|-------|--------|
| **Deaths from chronic inflammation** | 3 in 5 globally | [[01_Inflammation_Thesis]] |
| **US adults with elevated hs-CRP** | 34.6% | Population biomarker data |
| **Annual US healthcare spend** | $4.1 trillion | CMS national health expenditure data |
| **Downstream spend share** | 90% of total | Treating consequences, not causes |
| **Upstream interception spend** | ~0% | Almost nothing spent at origin nodes |

> The investment thesis: the healthcare system has a $3.7 trillion downstream spend problem with almost no competing solutions at the upstream origin nodes.

## Revenue Architecture

### Four Revenue Streams

| Stream | Product | Model | Margin Profile |
|--------|---------|-------|---------------|
| **1. Professional intervention** | Lytica via Maxoral | Per-procedure | High margin |
| **2. Consumer maintenance** | Loria subscription | Recurring D2C | Moderate margin, high LTV |
| **3. Gut restoration** | Magisnat + Biolumen + Monch Monch | Recurring D2C | Moderate margin |
| **4. Platform SaaS** | Fasolati Engine + SciAgent | Subscription | High margin (software) |

### Revenue Flow

```
PATIENT JOURNEY                    REVENUE

Diagnosis (periodontitis)
         │
         ▼
Lytica Treatment (in-office)  →→→  HIGH-MARGIN PROCEDURE
         │                         (Maxoral channel)
         ▼
Loria Subscription (daily)    →→→  RECURRING D2C REVENUE
         │                         (cancel anytime)
         ▼
Gut Stack Added (protocol)    →→→  RECURRING D2C REVENUE
         │                         (bundled subscription)
         ▼
Engine Subscription (ongoing) →→→  SAAS REVENUE
         │                         (platform + AI)
         ▼
Retest / Retreatment (6-12mo) →→→  HIGH-MARGIN RE-PROCEDURE
```

## DSO Partnership Model

> DSOs are the primary B2B growth channel. They provide patient volume, clinical infrastructure, and a recurring relationship.

### Value Proposition for DSOs

| DSO Benefit | Detail |
|-------------|--------|
| **New revenue line** | Lytica as a billable in-office procedure |
| **Patient retention** | Loria subscription creates ongoing patient relationship |
| **Differentiation** | Oral-systemic integrated care as competitive advantage |
| **Data** | Fasolati Engine provides outcome data across the network |
| **Full system of care** | Not a single product but a complete therapeutic cascade |

### DSO Economics

| Metric | Component |
|--------|-----------|
| **Per-procedure revenue** | Lytica intervention (Stage III-IV perio, peri-implantitis) |
| **Per-patient-per-month** | Loria subscription + Gut Stack + Engine |
| **Network data value** | Aggregated outcome data across all clinics |
| **Supply chain** | Henry Schein / Patterson / Benco fulfillment |

## Node Network Thesis

> "Scale the node network across chronic disease prevention"

The Fasolati "node network" is the thesis that as more clinics, patients, and data points connect to the Fasolati Engine, the platform's value compounds:

| Growth Vector | Mechanism |
|--------------|-----------|
| **More clinics** | More intervention procedures, more subscriptions |
| **More patients** | More data for SciAgent model refinement |
| **More data** | Better personalization, better outcomes, better evidence |
| **Better outcomes** | Stronger clinical evidence, faster regulatory clearance |
| **Stronger evidence** | More clinics adopt, more patients join |

```
        Clinics ──→ Patients ──→ Data
           ▲                       │
           │                       ▼
        Evidence ◄── Outcomes ◄── Models
```
