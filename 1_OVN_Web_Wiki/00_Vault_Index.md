# OVN Nexus Vault — Master Index

## What This Vault Is

A comprehensive internal reference for every section, feature, data model, and claim on **ovnnexus.com** — the Oral-Vascular-Neural Research Platform built by **S. Thaddeus Connelly, DDS, MD, PhD, FACS** (UCSF / SFVAMC / GengyeUSA).

This vault captures the full scope of the platform: public-facing educational pages, the member-only research hub, backend data infrastructure, and administrative tooling. Every file is written in clear paraphrased prose with Obsidian `[[wiki-links]]` throughout.

## How to Use This Vault

1. **Open in Obsidian** → "Open folder as vault" → point to `1_OVN_Web_Wiki/`.
2. **Navigate via links** — every `[[bracketed reference]]` is clickable.
3. **Add your own notes** — commentary, literature tags, Excalidraw diagrams, PDF attachments.
4. **Settings recommendation** → Files & Links → enable "Detect all file changes" for a living-document feel.
