# Showcase – Gingival Immunity v2.0: The Wiring Diagram

> Route: `/showcase`

---

## What This Presentation Is

A **cinematic, molecularly resolved, spatially and dynamically controlled model of gingival immunity** designed for teaching, research communication, and clinical storytelling. The full title is "Gingival Immunity v2.0: The Wiring Diagram" and it comprises **40 slides** across **5 visual templates** with integrated AI narration and ambient music.

This is not a standard slide deck — it is a multimedia research communication tool built on top of modern AI infrastructure.

---

## 5 Presentation Templates

Each template reframes the same 40-slide scientific content in a different visual and narrative tone:

| Template | Style | Best For |
|----------|-------|----------|
| **Documentary** | Clean, journalistic, neutral palette | Broad audiences, public lectures, media |
| **Cinematic Dark** | Dramatic lighting, deep contrast, high visual impact | Keynotes, high-stakes scientific storytelling |
| **Modern Minimal** | Sleek, uncluttered, generous whitespace | Conferences, peer review presentations |
| **Science Journal** | Publication-ready aesthetic, detailed call-outs, figure numbering | Journal submissions, grant applications |
| **Impact Story** | Narrative-driven, links molecular events to patient outcomes | Patient advocacy, clinical education, fundraising |

## 40-Slide Architecture

The presentation follows a logical progression from tissue-level anatomy through molecular mechanisms to clinical implications:

| Slide Range | Theme | Key Content |
|-------------|-------|-------------|
| 1–3 | Introduction | Title, OVN Axis overview, gingival anatomy primer |
| 4–8 | Tissue Architecture | Junctional epithelium, sulcular epithelium, connective tissue layers, vascular supply |
| 9–14 | Innate Immunity | Neutrophil surveillance, epithelial barrier function, antimicrobial peptides, complement activation |
| 15–20 | Adaptive Immunity | T-cell responses, B-cell/antibody dynamics, antigen presentation in gingival tissue |
| 21–26 | Microbial Challenge | Biofilm formation, keystone pathogen (*P. gingivalis*) virulence, OMV release and cargo |
| 27–32 | The OVN Cascade | 5-step cascade at tissue resolution — barrier breach through tissue endpoints |
| 33–37 | Systemic Connections | Vascular dissemination, BBB penetration, cardiac tissue effects, immune modulation |
| 38–40 | Clinical Synthesis | Evidence tiers recap, "What You Can Do Today", future research directions |

Each slide can be expanded with personal commentary, literature links, and clinical pearls.

## AI Video Narration Pipeline

The showcase includes a sophisticated automated video production workflow:

### Pipeline Architecture

```
Slide Images → GPT-4o Vision Analysis → Script Generation
    → ElevenLabs / OpenAI Voice Narration → Ambient Music Synthesis
    → Final Composition → Supabase CDN Storage
```

### Pipeline Components

| Component | Technology | Function |
|-----------|-----------|----------|
| Slide extraction | Image processing | Convert each slide to high-resolution image |
| Visual analysis | GPT-4o Vision | Analyze slide content to generate accurate narration scripts |
| Voice narration | ElevenLabs / OpenAI TTS | Multiple narrator profiles with selectable voice characteristics |
| Ambient music | AI synthesis | Background audio matched to presentation tone and template |
| Storage & delivery | Supabase CDN | Reliable hosting with automated retry logic |

### UX Features

- **Upload flow** — drag-and-drop slide deck upload
- **Narrator profiles** — multiple AI voices with distinct characteristics
- **Teaching-style selectors** — adjust narration complexity and pace
- **Target-audience presets** — clinician, researcher, student, general public
- **Automated retry logic** — handles CDN upload failures gracefully
