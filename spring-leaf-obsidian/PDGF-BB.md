# PDGF-BB — Platelet-Derived Growth Factor BB

Signal #2 in the [[4-Signal Construct]]. Supports vascular network formation along neural pathways established by [[NGF]].

## Role in the Construct

- **Function:** Vascular support — recruits pericytes, stabilizes new vessels
- **Phase:** VESSELS (second phase of the [[Neural-First Paradigm]] cascade)
- **Mechanism:** Angiogenesis along neural pathways, mesenchymal cell recruitment
- **Criticality:** Without vascular support, regenerating tissue cannot sustain itself

## Biological Function

PDGF-BB is a potent mitogen and chemoattractant:
- Stimulates angiogenesis and vessel maturation
- Recruits mesenchymal stem cells
- Promotes wound healing
- Indirect neurotrophic effects (the basis for [[BioHorizons - GEM 21S|GEM 21S]]'s partial efficacy)

## Cascade Position

```
[[NGF]] → Neural ingrowth
   ↓
[PDGF-BB] → Vascular network
   ↓
[[BMP-7]] → PDL differentiation
   ↓
[[BDNF]] → Neural maintenance
```

## Existing Clinical Use

PDGF-BB is the active ingredient in [[BioHorizons - GEM 21S|GEM 21S]] (combined with β-TCP). GEM 21S has demonstrated some efficacy but lacks the full [[Neural-First Paradigm]] cascade — it addresses vasculature but not neural ingrowth directly.

This validates PDGF-BB's role while highlighting the need for the complete [[4-Signal Construct]].
