# Manufacturing Plan

[[Spring Leaf Bio]]'s phased manufacturing strategy for the [[4-Signal Construct]], progressing from contract manufacturing to full vertical integration.

## COGS Progression

```
Phase 1: $180/unit
Phase 2: $140/unit  (-22%)
Phase 3:  $85/unit  (-53%)
```

## Phase 1: Contract Manufacturing (Month 0–24)

| Parameter | Detail |
|-----------|--------|
| **Model** | CMO partner |
| **Setup cost** | $200–300K |
| **COGS** | $180/unit |
| **Batch size** | 500 units |
| **Annual capacity** | 10,000 units |

- Leverages existing CMO infrastructure
- Minimizes upfront capital
- Funded by [[Funding|$3.5M seed round]] (20% = $700K for manufacturing)

## Phase 2: Hybrid Model (Month 24–48)

| Parameter | Detail |
|-----------|--------|
| **Model** | In-house QC + CMO bulk manufacturing |
| **Setup cost** | $1.5–2M |
| **COGS** | $140/unit |
| **Batch size** | 1,000 units |
| **Annual capacity** | 50,000 units |

- Quality control brought in-house
- 5× capacity increase
- Supports [[Three-Phase Commercialization|Phase 2]] cell-based product

## Phase 3: Vertical Integration (Month 48+)

| Parameter | Detail |
|-----------|--------|
| **Model** | Full in-house manufacturing |
| **Setup cost** | $8–12M |
| **COGS** | $85/unit |
| **Batch size** | 5,000 units |
| **Annual capacity** | 500,000 units |

- Complete control over process and quality
- 50× capacity vs. Phase 1
- In-house growth factor production (see [[Supply Chain]])
- Drives [[Business Model|Phase 3 gross margin to 82%]]
